Simple HTML + JavaScript Flappy Bird clone. To run locally:

Open `index.html` in a browser, or run a simple static server from the project folder:

```bash
python3 -m http.server 8000
# then open http://127.0.0.1:8000 in your browser
```

Controls: press Space or click/tap the canvas to flap. Click to restart after Game Over.
